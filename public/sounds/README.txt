Sound Files Required
====================

Place the following MP3 files in this directory:

1. bowl-start.mp3   - Singing bowl sound for timer start
2. bowl-complete.mp3 - Singing bowl sound for session complete
3. tick.mp3         - Optional soft tick sound

Recommended sources for royalty-free sounds:
- freesound.org
- pixabay.com/sound-effects
- zapsplat.com
