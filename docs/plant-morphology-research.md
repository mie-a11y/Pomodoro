# 盆栽形态研究参考

## 日式盆景 (Bonsai) 核心形态特征

### 1. 主干特征

| 特征 | 描述 | 几何实现 |
|------|------|----------|
| **自然弯曲** | 主干从不完全垂直，总有轻微倾斜 | rotation.z: 3-8° |
| **锥形** | 根部粗壮，向上逐渐变细 | radiusBottom > radiusTop (1.5:1) |
| **向光性** | 倾斜方向通常朝向假想光源 | 固定向左或向右倾斜 |
| **树皮纹理** | 粗糙、有年轮感 | roughness: 0.85-0.95 |

### 2. 枝条特征

| 特征 | 描述 | 几何实现 |
|------|------|----------|
| **交替生长** | 枝条不从同一高度对称长出 | 高度: 40%, 60%, 80% 交错 |
| **向外向上** | 枝条向外展开同时略向上 | rotationX: 10-30°, rotationZ: 30-60° |
| **不对称** | 左右枝条数量、长度不同 | 左侧 2 根，右侧 1 根 |
| **渐细** | 枝条比主干细，末端更细 | radius: 主干的 20-30% |

### 3. 叶片特征

| 特征 | 描述 | 几何实现 |
|------|------|----------|
| **扁平** | 叶子是片状而非球状 | PlaneGeometry |
| **朝向多样** | 每片叶子朝向不同 | 随机 pitch/yaw/roll |
| **末端聚集** | 叶子长在枝条末端 | 绑定到 branch 末端 |
| **颜色变化** | 新叶嫩绿，老叶深绿 | HSL 亮度 ±10% |

### 4. 整体形态规则

```
         *  *          <- 叶片（不规则分布）
        * ** *
         \|/           <- 枝条（向外展开）
      *   |  *
       \  |
        \ |  /         <- 主干（轻微倾斜）
         \| /
          |
         /|\           <- 另一侧枝条
        / |
       ===|===         <- 花盆（稳定基座）
```

### 5. 关键的"不完美"

真实植物的"不完美"特征：

1. **非对称**：左右看起来不一样
2. **非均匀**：间距不等，角度不等
3. **非完美**：允许"长歪"的枝条
4. **非机械**：没有精确的 45° 或 90°

### 6. 各阶段形态参考

| 阶段 | 主干 | 枝条 | 叶片 | 整体印象 |
|------|------|------|------|----------|
| sprout | 短、细、微倾 | 无 | 顶部 1-2 片小叶 | 刚发芽的小苗 |
| young | 中等、锥形明显 | 1 根主枝 | 3-4 片 | 开始有形状 |
| mature | 较高、粗壮 | 2 根枝条 | 5-6 片 | 有盆栽感 |
| ancient | 完整高度 | 3 根枝条 | 7-8 片 | 成熟盆景 |

## 与当前实现的对比

### 当前问题 ❌

1. 使用 SphereGeometry 作为树冠 → **违反"叶子是片"原则**
2. 主干完全垂直 → **缺乏自然倾斜**
3. 枝条也是球形 → **不像枝条**
4. 完美对称布局 → **太机械化**

### 目标改进 ✅

1. 叶片使用 PlaneGeometry，扁平展开
2. 主干添加 3-8° 倾斜
3. 枝条使用细长 CylinderGeometry
4. 整体不对称，左右不同
